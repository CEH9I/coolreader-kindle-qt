1. Place Kindlet-2.2.jar from your Kindle into /lib
2. Run "gradlew" to build Cool Reader runner
3. Run "gradlew -P TouchRunner" to build generic Touch Runner
4. For Touch Runner copy /conf/touchrunner directory to Kindle
5. Cool Reader runner can optionally use commands.txt from /cr3
6. Place TouchRunner-3.0.azw2 and/or cr3runner-3.0.azw2 from /build/libs
   into /documents on Kindle
